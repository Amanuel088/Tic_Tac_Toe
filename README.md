# Tic Tac Toe

A command-line Tic Tac Toe game built with Ruby.

## About

Two human players take turns on a 3x3 grid, trying to get three markers in a row — horizontally, vertically, or diagonally. The game detects wins and draws automatically.

## How to Run

Make sure Ruby is installed, then:
ruby main.rb
